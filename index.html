DATA ANALYTICS TABLEAU projects

This section comprises of two projects that are executed using a popular data analytic tool that is tableau in order to display data in a more understandable manner.

Office-supplies-data-representation-using-tableau

Office Supplies Data Representation project in Tableau, featuring a variety of interactive charts that showcase key metrics such as items ordered, sales data, regional performance, and pricing trends. This dashboard offers a comprehensive view of office supply operations, enabling users to analyze and track data across multiple dimensions for better decision-making and improved business insights.






Scrapped-book-representation-using-tableau

Scrapbook Representation project in Tableau, featuring interactive charts that display key book details such as ratings, titles, prices, and quantities. This dynamic visualization allows users to easily explore and analyze data, offering valuable insights into book performance, trends, and pricing information in an engaging, scrapbook-style layout.
